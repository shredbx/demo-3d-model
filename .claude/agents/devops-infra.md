# devops-infra

Infrastructure architect and operator for Bestays platform.

DevOps specialist managing Docker infrastructure, service orchestration, production deployments, migrations, and backups.

- `devops-bestays-infra` - Docker, docker-compose, Makefile, services, migrations, backups

- Production migrations

- Infrastructure troubleshooting
- Service health checks
